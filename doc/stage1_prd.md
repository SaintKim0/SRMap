# Stage 1 PRD: SceneMap MVP

## 1. 프로젝트 개요

### 1.1 목적
드라마/영화 촬영지 정보를 제공하여 K-콘텐츠 팬들의 '세트 제팅(Set-jetting)' 경험을 지원하는 모바일 앱의 MVP 버전 개발

### 1.2 목표
- 특정 인기 드라마 1-2개의 촬영지 정보 제공
- 지도 기반 촬영지 탐색 기능
- 기본적인 검색 및 필터링 기능
- 2개월 내 MVP 출시

### 1.3 타겟 사용자
- K-드라마/영화 팬 (국내외)
- 20-40대 여성 (주 타겟)
- 촬영지 방문을 계획하는 관광객
- '인생샷'을 원하는 SNS 활동 사용자

---

## 2. MVP 핵심 기능

### 2.1 홈 화면
**목적**: 사용자에게 주요 촬영지를 빠르게 소개

#### 기능 요구사항
- [ ] 인기 촬영지 TOP 10 표시 (카드 형태)
- [ ] 최근 추가된 촬영지 섹션
- [ ] 검색 바 (상단 고정)
- [ ] 하단 네비게이션 바 (홈, 지도, 검색, 마이페이지)

#### UI 요소
- 촬영지 카드: 대표 이미지, 촬영지명, 드라마명, 거리 정보
- 스크롤 가능한 수평/수직 리스트
- 로딩 인디케이터

---

### 2.2 촬영지 목록 화면
**목적**: 모든 촬영지를 탐색하고 필터링

#### 기능 요구사항
- [ ] 전체 촬영지 리스트 표시
- [ ] 리스트/그리드 뷰 전환
- [ ] 정렬 기능
  - 거리순 (가까운 순)
  - 인기순 (조회수/좋아요)
  - 최신순
- [ ] 필터링 기능
  - 드라마/영화별
  - 지역별 (서울, 부산, 제주 등)
  - 카테고리별 (카페, 공원, 건물 등)

#### UI 요소
- 필터 버튼 (상단)
- 정렬 드롭다운
- 촬영지 카드 (리스트/그리드)
- 무한 스크롤 또는 페이지네이션

---

### 2.3 촬영지 상세 화면
**목적**: 촬영지의 상세 정보 제공

#### 기능 요구사항
- [ ] 이미지 갤러리 (스와이프 가능)
  - 촬영지 실제 사진 3-5장
  - 드라마 장면 참고 이미지 (저작권 고려)
- [ ] 기본 정보
  - 촬영지명
  - 주소
  - 영업 시간 (해당 시)
  - 전화번호
  - 웹사이트
- [ ] 관련 드라마/영화 정보
  - 작품명
  - 장면 설명
  - 회차 정보
- [ ] 지도 미리보기 (작은 지도)
- [ ] 액션 버튼
  - 길찾기 (외부 지도 앱 연동)
  - 공유하기
  - 북마크 (저장)
- [ ] 교통 정보
  - 대중교통 안내
  - 주차 정보

#### UI 요소
- 이미지 캐러셀
- 정보 섹션 (카드 형태)
- 고정 하단 액션 바

---

### 2.4 지도 화면
**목적**: 지도에서 촬영지 위치 확인 및 탐색

#### 기능 요구사항
- [ ] 지도에 촬영지 마커 표시
- [ ] 현재 위치 표시
- [ ] 마커 클릭 시 간단한 정보 표시 (바텀시트)
- [ ] 지도 확대/축소
- [ ] 주변 촬영지 검색 (현재 지도 영역 내)
- [ ] 필터 적용 (드라마별, 카테고리별)

#### UI 요소
- 전체 화면 지도
- 커스텀 마커 (드라마 아이콘 또는 카테고리 아이콘)
- 바텀시트 (촬영지 정보)
- 필터 버튼 (플로팅)
- 현재 위치 버튼

---

### 2.5 검색 화면
**목적**: 원하는 촬영지 빠르게 찾기

#### 기능 요구사항
- [ ] 통합 검색
  - 드라마/영화명으로 검색
  - 촬영지명으로 검색
  - 지역명으로 검색
- [ ] 검색 결과 표시
- [ ] 최근 검색어 저장
- [ ] 인기 검색어 표시
- [ ] 자동완성 (선택사항)

#### UI 요소
- 검색 입력 필드
- 최근/인기 검색어 칩
- 검색 결과 리스트
- 빈 상태 화면 (검색 결과 없음)

---

### 2.6 마이페이지 (기본)
**목적**: 사용자 설정 및 저장된 정보 관리

#### 기능 요구사항
- [ ] 북마크한 촬영지 목록
- [ ] 설정
  - 테마 변경 (라이트/다크)
  - 알림 설정
  - 언어 설정 (한국어/영어)
- [ ] 앱 정보
  - 버전 정보
  - 이용약관
  - 개인정보 처리방침

#### UI 요소
- 프로필 섹션 (간단한 아바타)
- 메뉴 리스트
- 설정 토글/선택

## 3. 데이터 모델

### 3.1 Location (촬영지)
```dart
class Location {
  String id;              // 고유 ID
  String name;            // 촬영지명
  String address;         // 주소
  double latitude;        // 위도
  double longitude;       // 경도
  String category;        // 카테고리 (카페, 공원 등)
  String? phoneNumber;    // 전화번호
  String? website;        // 웹사이트
  String? openingHours;   // 영업시간
  List<String> imageUrls; // 이미지 URL 목록
  String? parking;        // 주차 정보
  String? transportation; // 대중교통 정보
  int viewCount;          // 조회수
  int bookmarkCount;      // 북마크 수
  DateTime createdAt;     // 생성일
}
```

### 3.2 Content (드라마/영화)
```dart
class Content {
  String id;              // 고유 ID
  String title;           // 작품명
  String type;            // 드라마 or 영화
  String? posterUrl;      // 포스터 이미지
  int? releaseYear;       // 방영/개봉 연도
  String? genre;          // 장르
  String? description;    // 설명
}
```

### 3.3 Scene (장면)
```dart
class Scene {
  String id;              // 고유 ID
  String locationId;      // 촬영지 ID (FK)
  String contentId;       // 드라마/영화 ID (FK)
  String description;     // 장면 설명
  String? episode;        // 회차 정보
  String? sceneImageUrl;  // 장면 이미지 (참고용)
}
```

### 3.4 User (사용자 - 기본)
```dart
class User {
  String id;              // 고유 ID
  List<String> bookmarks; // 북마크한 촬영지 ID 목록
  String theme;           // 테마 설정
  String language;        // 언어 설정
}
```

## 4. 기술 요구사항

### 4.1 필수 패키지
- ✅ `provider`: 상태 관리
- ✅ `go_router`: 라우팅
- ✅ `sqflite`: 로컬 데이터베이스
- ✅ `shared_preferences`: 사용자 설정 저장
- ✅ `google_fonts`: 폰트
- ⚠️ `google_maps_flutter` 또는 `flutter_naver_map`: 지도 (추가 필요)
- ⚠️ `geolocator`: 위치 서비스 (추가 필요)
- ⚠️ `url_launcher`: 외부 링크/앱 실행 (추가 필요)

### 4.2 권한 설정
- 위치 권한 (현재 위치 표시)
- 인터넷 권한 (지도 및 이미지 로드)

## 5. MVP 범위 제한

### 5.1 포함하지 않는 기능 (추후 단계)
- ❌ 사용자 로그인/회원가입
- ❌ 리뷰 작성
- ❌ 사진 업로드
- ❌ AR 오버레이
- ❌ 스탬프 투어
- ❌ 코스 추천
- ❌ 실시간 정보 업데이트

### 5.2 데이터 범위
- 인기 드라마 1-2개 (예: "도깨비", "이태원 클라쓰")
- 촬영지 20-30개소
- 서울/경기 지역 중심
