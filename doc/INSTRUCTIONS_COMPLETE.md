# 맛있는녀석들 전체 데이터 수집 방법

## 현재 상황
✓ 스크립트가 정상 작동합니다!
✓ 현재 HTML 파일에서 **20개의 식당 데이터**를 성공적으로 추출했습니다.
⚠️ 하지만 전체 **122개의 식당** 중 20개만 저장되어 있습니다.

## 전체 데이터를 수집하는 방법

### 방법 1: 페이지를 스크롤하여 모든 데이터 로드 후 저장

1. **브라우저에서 페이지 열기**
   ```
   https://www.diningcode.com/list.dc?query=%EA%B2%BD%EA%B8%B0%EB%8F%84%20%EB%A7%9B%EC%9E%88%EB%8A%94%EB%85%80%EC%84%9D%EB%93%A4
   ```

2. **페이지 끝까지 스크롤**
   - 페이지를 천천히 아래로 스크롤하세요
   - 새로운 식당들이 계속 로드됩니다
   - **122개 모두 표시될 때까지** 스크롤하세요
   - 페이지 하단에 "더보기" 버튼이 있으면 클릭하세요

3. **HTML 파일로 저장**
   - 모든 데이터가 로드되면
   - `Ctrl + S` (또는 우클릭 → 다른 이름으로 저장)
   - "웹페이지, 완료" 형식으로 저장
   - 기존 파일을 덮어쓰세요

4. **스크립트 실행**
   ```bash
   python extract_from_saved_html.py
   ```

### 방법 2: 개발자 도구 콘솔 사용 (더 쉬운 방법)

1. **브라우저에서 페이지 열고 스크롤**
   - 위 방법 1의 1-2단계 수행
   - 122개 모두 로드될 때까지 스크롤

2. **개발자 도구 열기**
   - `F12` 키를 누르거나
   - 우클릭 → 검사

3. **Console 탭으로 이동**

4. **다음 코드 복사 & 붙여넣기 후 Enter**
   ```javascript
   copy(JSON.stringify(JSON.parse(localStorage.getItem('listData')).poi_section.list, null, 2))
   ```

5. **데이터가 클립보드에 복사됨**
   - 메모장이나 텍스트 에디터를 열고
   - `Ctrl + V`로 붙여넣기
   - `restaurants_data.json` 파일로 저장

6. **Python 스크립트로 JSON 파일 처리**
   ```bash
   python process_json_data.py
   ```
