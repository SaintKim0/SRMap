[C vs V] "식당을 고를 때 '프라이빗 룸'이 필수인가요, 아니면 '북적이는 활기'가 좋은가요?"

[S vs H] "전문가 평점이 높은 곳 vs 요즘 유튜브나 방송에서 가장 많이 언급되는 곳?"

[C vs N] "대대손손 내려오는 원조 맛집 vs 이전에 없던 새로운 조합의 요리?"

💡 기획 포인트
이렇게 4가지 지표로 나누면 총 **16가지(2x2x2x2)**의 성향이 나옵니다. 앱 내 커뮤니티에서 "P.V.H.N 모여라!" 같은 소모임을 만들면 취향이 비슷한 사람들끼리 메뉴 선정부터 웨이팅 공유까지 훨씬 활발하게 일어날 것입니다.

이 16가지 결과에 대해, 사용자가 앱에서 볼 수 있는 '캐릭터 카드'와 '찰떡궁합 메뉴'를 디자인해 볼까요?

잘했어. 그런ㄷ, 이걸 조금 더 세분화 하자. 애매하게 반쯤걸친 성향들도 있으니까 말이야. 알겠니?

성향이 칼로 자르듯 나뉘지 않고 **'중간 지대'**나 **'복합적 성향'**을 가진 유저들을 위해, 지표를 고정된 값이 아닌 **스펙트럼(점수제)**으로 설계하고 이를 **'성격의 농도'**로 표현하는 방식을 제안합니다.

단순히 P냐 E냐가 아니라, "실속형에 가까운 프리미엄" 같은 세분화된 결과가 가능하도록 지표를 고정(Fixed)형에서 비중(Weight)형으로 확장하는 것입니다.

🥗 세분화된 맛집 MBTI: P.E.A.T v2.0 (The Spectrum)
단순 선택지가 아닌 **"어느 쪽에 더 가까운가?"**를 5단계(매우 그렇다 ~ 매우 아니다)로 측정하여 4가지 지표의 '비중'을 조합합니다.

1. 4가지 핵심 지표의 확장 (Spectrum Scales)
Price (지갑 가드)

P (Premium): "한 끼에 20만 원 이상도 가치 있다면 OK"

M (Moderate): "가격과 맛의 밸런스가 가장 중요 (중간층)" -> 신설

E (Economy): "가성비가 최고의 미덕"

Environment (공간 온도)

C (Calm): "조용한 대화와 프라이빗함이 필수"

B (Balanced): "적당히 활기차면서도 대화가 가능한 곳" -> 신설

V (Vivid): "북적거리고 사람 냄새 나는 곳이 진짜 맛집"

Authority (신뢰의 출처)

S (Specialist): "미슐랭/블루리본 등 전문가 권위를 절대 신뢰"

O (Open-minded): "전문가도 좋고 예능도 좋고, 두루 참고" -> 신설

H (Hype): "방송, 유튜브, 흑백요리사 등 대중적인 화제성 중시"

Taste (맛의 뿌리)

C (Classic): "원조, 오리지널리티, 변하지 않는 깊은 맛"

F (Fusion/Creative): "셰프의 개성, 새로운 식재료 조합, 실험적 맛"

2. 세분화된 '애매한 성향'들을 위한 결과 타입 예시
지표의 비중에 따라 **'하이브리드 유형'**을 명칭으로 부여합니다.

"실속 있는 미식가" (M.C.S.C)

특성: 미슐랭 스타보다는 빕 구르망이나 Registered 레스토랑을 선호. 가격은 합리적이어야 하지만 전문가의 검증은 포기 못 하며, 조용한 곳을 찾는 스타일.

"트렌디한 노포 마니아" (E.V.O.C)

특성: 예능에 나온 노포를 좋아하지만, 맛은 전통적이어야 함. 왁자지껄한 분위기를 즐기며 흑백요리사의 '시장 장인' 같은 스타일에 열광함.

"화려한 미식 탐험가" (P.B.H.F)

특성: 돈을 아끼지 않고 흑백요리사 식당이나 화제의 컨템퍼러리 다이닝을 찾아다님. 너무 정적인 것보다는 활기찬 셰프의 퍼포먼스를 즐김.

3. 설문 조사 방식의 디테일 (8:2 혹은 7:3 비중)
질문 결과가 5:5에 가깝게 나오면 "중립(Neutral)" 혹은 "올라운더(All-rounder)" 칭호를 부여하여 유연성을 줍니다.

질문 예시: "미슐랭 식당과 예능 맛집 중 어디로 갈까요?"

답변 A: 무조건 미슐랭 (S +2)

답변 B: 미슐랭을 더 선호하지만 예능 맛집도 궁금 (S +1)

답변 C: 둘 다 상관없음 (O)

답변 D: 예능 맛집을 더 선호하지만 미슐랭도 가끔 (H +1)

답변 E: 무조건 예능 맛집 (H +2)

💡 기획 포인트: 커뮤니티 연결
이렇게 세분화하면 "P.C.S.C(극강의 럭셔리)" 그룹과 "M.C.S.C(실속형 럭셔리)" 그룹이 같은 채널에서 대화하되, 서로 다른 가격대의 정보를 교환하며 시너지를 낼 수 있습니다.

이제 이 세분화된 지표를 기반으로, 각 유저들에게 보여줄 '미식 리포트'와 '오늘의 추천 메뉴' 로직을 짜 드릴까요? 예를 들어 "당신은 가성비 70%, 도전정신 30%의 미식가입니다" 같은 형태입니다.

[PRD] 미식 DNA(P.E.A.T) 기반 초개인화 맛집 추천 시스템
1. 제품 개요 (Product Overview)
제품명: (가칭) TastyMap - 초개인화 미식 큐레이션 서비스

핵심 가치: TV 예능, 흑백요리사, 미쉐린 등 방대한 맛집 데이터를 사용자의 미식 성향(P.E.A.T)과 결합하여 "실패 없는 오늘 점심"을 제안함.

목표: 단순한 지도 앱을 넘어, 사용자 취향 분석을 통한 커뮤니티 형성 및 충성 고객 확보.

2. 주요 기능 및 사용자 여정 (Key Features)
Onboarding (DNA 테스트): 설치 직후 4~5개 문항의 설문을 통해 유저의 P.E.A.T 유형 결정.

Adaptive Map (맞춤형 지도): DNA 결과에 따라 지도상의 핀(Pin) 노출 우선순위 조정.

Personalized Theme (오늘은 뭐 먹지?): 사용자의 예산과 기분을 반영한 매일 오전 11:30 맞춤 푸시.

Community (성향별 모임): 동일한 P.E.A.T 코드를 가진 유저들끼리의 리뷰 공유 및 번개 모임.

플러터(Flutter)에서 즉시 실행 가능한 미식 DNA 테스트(P.E.A.T Survey) 설문 페이지 전체 소스 코드입니다.

사용자가 선택지를 클릭할 때마다 부드러운 애니메이션 효과를 주며, 4가지 지표(Price, Energy, Authority, Taste)를 점수화하여 결과 페이지로 넘겨주는 로직까지 포함되어 있습니다.

🚀 TastyMap Survey Page Full Code
Dart
import 'package:flutter/material.dart';

// 디자인 테마 설정
class TastyTheme {
  static const Color primaryRed = Color(0xFFE62117);
  static const Color accentGold = Color(0xFFD4AF37);
  static const Color bgDark = Color(0xFF121212);
  static const Color surfaceDark = Color(0xFF1E1E1E);
}

void main() => runApp(MaterialApp(
      theme: ThemeData.dark().copyWith(scaffoldBackgroundColor: TastyTheme.bgDark),
      home: const SurveyPage(),
    ));

class SurveyPage extends StatefulWidget {
  const SurveyPage({super.key});

  @override
  State<SurveyPage> createState() => _SurveyPageState();
}

class _SurveyPageState extends State<SurveyPage> {
  int _currentStep = 0;
  
  // 지표별 점수 (P, E, A, T)
  final Map<String, double> _scores = {'P': 0, 'E': 0, 'A': 0, 'T': 0};

  // 설문 데이터 구성
  final List<Map<String, dynamic>> _questions = [
    {
      'dimension': 'P',
      'question': '기념일 저녁, 당신의 선택은?',
      'options': [
        {'text': '가격 상관없이 최고급 다이닝', 'value': 1.0, 'img': 'https://images.unsplash.com/photo-1514362545857-3bc16c4c7d1b?w=400'},
        {'text': '합리적인 가격의 빕 구르망', 'value': -1.0, 'img': 'https://images.unsplash.com/photo-1552611052-33e04de081de?w=400'},
      ]
    },
    {
      'dimension': 'E',
      'question': '선호하는 식당의 분위기는?',
      'options': [
        {'text': '고요하고 프라이빗한 공간', 'value': 1.0, 'img': 'https://images.unsplash.com/photo-1517248135467-4c7edcad34c4?w=400'},
        {'text': '활기차고 시끌벅적한 노포', 'value': -1.0, 'img': 'https://images.unsplash.com/photo-1555396273-367ea4eb4db5?w=400'},
      ]
    },
    {
      'dimension': 'A',
      'question': '맛집을 고르는 기준은?',
      'options': [
        {'text': '미쉐린 등 전문가의 검증', 'value': 1.0, 'img': 'https://images.unsplash.com/photo-1581349485608-9469926a8e5e?w=400'},
        {'text': 'TV 예능이나 SNS 핫플', 'value': -1.0, 'img': 'https://images.unsplash.com/photo-1504674900247-0877df9cc836?w=400'},
      ]
    },
    {
      'dimension': 'T',
      'question': '오늘 끌리는 요리 스타일은?',
      'options': [
        {'text': '전통을 지킨 원조의 맛', 'value': 1.0, 'img': 'https://images.unsplash.com/photo-1547592166-23ac45744a05?w=400'},
        {'text': '셰프의 창의적인 퓨전 요리', 'value': -1.0, 'img': 'https://images.unsplash.com/photo-1512621776951-a57141f2eefd?w=400'},
      ]
    },
  ];

  void _handleAnswer(double value) {
    setState(() {
      String dim = _questions[_currentStep]['dimension'];
      _scores[dim] = value;
      
      if (_currentStep < _questions.length - 1) {
        _currentStep++;
      } else {
        _showResult();
      }
    });
  }

  void _showResult() {
    // 결과 코드 계산 (양수면 앞글자, 음수면 뒷글자 성향)
    String resP = _scores['P']! > 0 ? 'P' : 'E';
    String resE = _scores['E']! > 0 ? 'C' : 'V';
    String resA = _scores['A']! > 0 ? 'S' : 'H';
    String resT = _scores['T']! > 0 ? 'C' : 'N';
    String finalCode = "$resP.$resE.$resA.$resT";

    showDialog(
      context: context,
      barrierDismissible: false,
      builder: (ctx) => AlertDialog(
        backgroundColor: TastyTheme.surfaceDark,
        title: const Text("미식 DNA 분석 완료!", textAlign: TextAlign.center),
        content: Column(
          mainAxisSize: MainAxisSize.min,
          children: [
            const Text("당신의 유형은:"),
            const SizedBox(height: 10),
            Text(finalCode, style: const TextStyle(fontSize: 32, fontWeight: FontWeight.bold, color: TastyTheme.accentGold)),
          ],
        ),
        actions: [
          TextButton(onPressed: () => Navigator.pop(ctx), child: const Text("지도 확인하기", style: TextStyle(color: TastyTheme.primaryRed)))
        ],
      ),
    );
  }

  @override
  Widget build(BuildContext context) {
    var q = _questions[_currentStep];
    
    return Scaffold(
      appBar: AppBar(
        backgroundColor: Colors.transparent,
        elevation: 0,
        title: Text("질문 ${_currentStep + 1} / ${_questions.length}", style: const TextStyle(color: TastyTheme.accentGold)),
        centerTitle: true,
      ),
      body: Padding(
        padding: const EdgeInsets.all(24.0),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            const LinearProgressIndicator(
              value: 0.25 * 1, // 단계별로 조절 가능
              backgroundColor: TastyTheme.surfaceDark,
              color: TastyTheme.primaryRed,
            ),
            const SizedBox(height: 40),
            Text(q['question'], style: const TextStyle(fontSize: 24, fontWeight: FontWeight.bold, color: Colors.white)),
            const SizedBox(height: 40),
            Expanded(
              child: ListView.separated(
                itemCount: 2,
                separatorBuilder: (context, index) => const SizedBox(height: 20),
                itemBuilder: (context, index) {
                  var opt = q['options'][index];
                  return GestureDetector(
                    onTap: () => _handleAnswer(opt['value']),
                    child: Container(
                      height: 180,
                      decoration: BoxDecoration(
                        borderRadius: BorderRadius.circular(20),
                        image: DecorationImage(
                          image: NetworkImage(opt['img']),
                          fit: BoxFit.cover,
                          colorFilter: ColorFilter.mode(Colors.black.withOpacity(0.4), BlendMode.darken),
                        ),
                        border: Border.all(color: Colors.white10, width: 1),
                      ),
                      alignment: Alignment.center,
                      child: Text(
                        opt['text'],
                        textAlign: TextAlign.center,
                        style: const TextStyle(fontSize: 18, fontWeight: FontWeight.bold, color: Colors.white),
                      ),
                    ),
                  );
                },
              ),
            ),
          ],
        ),
      ),
    );
  }
}
