# 앱 분리 계획 (A안)

## 1. 프로젝트 구분

| 프로젝트 | 컨셉 | 포함 콘텐츠 |
|----------|------|--------------|
| **현재 폴더** (`01_ScreenMap_Backup`) | **촬영현장** — 영화·드라마·K-POP 촬영지 | 전체보기, 촬영현장(영화 / 드라마 / K-POP) |
| **새 프로젝트** (맛집지도) | **맛집지도** — 맛집·미슐렝·예능 촬영지 | 전체보기, 흑백요리사, 미슐렝 코리아, **예능 촬영현장** |

- **예능** 촬영현장은 대부분 맛집이므로 **맛집지도 앱**에 포함한다.

---

## 2. 현재 프로젝트 = 촬영현장 앱 (적용 내용)

- **메인 탭**: 전체보기, 촬영현장 (2개만)
- **촬영현장 서브탭**: 영화, 드라마, K-POP (예능 제거)
- **제거**: 흑백요리사, 미슐렝 코리아 섹터 및 관련 UI/필터
- **데이터**: `mediaType` = movie, drama, artist 만 사용 (show 제외)
- **홈 기본 콘텐츠**: 영화 / 드라마 / K-POP 카드만 (예능 카드 제거)

---

## 3. 맛집지도 앱 (새 프로젝트 생성 시)

- **메인 탭**: 전체보기, 흑백요리사, 미슐렝 코리아, **예능 촬영현장**
- **촬영현장** 탭은 없음. **예능**만 별도 탭/섹터로 두고, 예능 프로그램별 맛집 리스트
- **데이터**: mediaType = blackwhite, guide, **show** + 기타 맛집
- **제거**: 영화, 드라마, K-POP 전용 UI 및 기본 콘텐츠 카드

**생성 방법**: 이 프로젝트를 복사한 뒤  
- `SPLIT_PLAN.md` 참고하여 흑백요리사/미슐렝/예능만 남기고  
- 촬영현장(영화/드라마/K-POP) 관련 버튼·필터·카드 제거  
- 앱명·아이콘을 맛집지도용으로 변경

---

## 4. 데이터(CSV/DB) 공유

- 두 앱이 **같은 locations 테이블 구조**를 쓸 수 있음.  
- 촬영현장 앱: `mediaType` in (movie, drama, artist) 만 조회  
- 맛집지도 앱: `mediaType` in (show, blackwhite, guide) + 기타 맛집만 조회  
- CSV를 두 개로 나눌 수도 있고, 한 CSV에 두고 앱에서 필터만 나눠도 됨.

---

- [x] 촬영현장 서브탭에서 예능 제거 (영화, 드라마, K-POP만)
- [x] 홈 기본 콘텐츠에서 예능 카드 제거
- [x] Provider에서 흑백요리사/미슐렝 필터 제거, 촬영현장에서 예능(show) 제거

### 맛집지도 앱 (별도 프로젝트)

- [x] **제거**: 촬영현장 메인 탭, 촬영현장 서브탭(영화/드라마/K-POP), 홈의 영화/드라마/K-POP SectorCard

- [x] **추가**: "예능 촬영현장" 메인 탭 또는 섹터 (mediaType=show) — 예능 프로그램 촬영지(맛집 위주)
- [x] Provider: 촬영현장 분기 제거, 흑백요리사/미슐렝/예능(show) 필터만 유지
- [x] 앱명(예: TasteMap/맛집지도)·테마·아이콘 변경 (앱명·표시명 적용; 아이콘은 doc/AppIcon.png 등 별도 교체 가능)
